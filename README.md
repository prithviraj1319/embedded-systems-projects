# SPI and I2C Communication Demo

## Overview
This folder contains simple example sketches showing how SPI and I2C communication can be initialized and used in an Arduino-style embedded system.

## Included Examples
- `spi_master_demo.ino`: Sends bytes to a slave device over SPI
- `i2c_master_demo.ino`: Sends commands and reads data from an I2C device

## Skills Demonstrated
- Communication protocol basics
- Peripheral initialization
- Data transfer and debugging structure
